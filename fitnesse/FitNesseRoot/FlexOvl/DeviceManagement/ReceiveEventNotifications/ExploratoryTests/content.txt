!3 Receive notifications for unregistered devices

Create a new device in the device simulator, donot register the the device.
Send an event notification.

This event notification should be rejected at the OSP

!3 Receive notifications with a hih index

Select a registered device and send event notifications with a high index e.g 2333 or 9999


!4 Remark with device-simulator it is not possible to test multiple events and/or multiple indexes
